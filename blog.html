EKS Deployment using Terraform

Objective: Deploy production-ready EKS cluster.

terraform init
terraform apply
aws eks update-kubeconfig --region ap-south-1 --name my-eks
kubectl get nodes

Outcome: Successfully deployed scalable Kubernetes cluster.

Terraform Remote Backend Setup

terraform {
  backend "s3" {
    bucket         = "tf-state"
    key            = "eks/terraform.tfstate"
    region         = "ap-south-1"
    dynamodb_table = "tf-lock"
  }
}

Outcome: Enabled remote state locking for safe collaboration.
